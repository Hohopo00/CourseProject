## Реализованные операции

1. Показать все записи - Отображение полной информации о всех добавленных записях.
2. Добавить новую запись - Ручной ввод данных о новом репозитории через диалог с пользователем.
3. Поиск записей - Фильтрация записей по размеру репозитория между минимальным и максимальным зачением.
4. Сортировка данных - Упорядочивание записей по хэшу названия.
5. Сохранить в файл - Сохранение всех записей в бинарный файл для постоянного хранения.
6. Загрузить из файла - Чтение ранее сохраненных записей из бинарного файла.
7. Создать тестовые данные - Автоматическая генерация случайных записей для тестирования функционала.
8. Выход - Завершение работы программы с освобождением выделенной памяти.

## Пользовательские функции

## 1. char* getLicense(int license)

- **Назначение**: Преобразует номер лицензии в текстовое название
- **Параметры**:
  - ``license`` - числовой код лицензии из enum License
- **Возвращает**: строку с названием лицензии

## 2. char* getActivity(int activity)

- **Назначение**: Преобразует номер активности в текстовое описание
- **Параметры**:
  - ``activity`` - числовой код активности из enum Activity
- **Возвращает**: строку с описанием уровня активности

## 3. int showItem(DataItem item, int number)

- **Назначение**: Выводит информацию об одном проекте на экран
- **Параметры**:
  - ``item`` - структура с данными проекта
  - ``number`` - порядковый номер проекта
- **Возвращает**: всегда 1

## 4. int showAll(DataItem* items, int count)

- **Назначение**: Выводит информацию обо всех проектах на экран
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - количество записей в массиве
- **Возвращает**: количество отображенных записей (0 если пусто)

## 5. int addItem(DataItem* items, int* count, DataItem* newItem)

- **Назначение**: Добавляет новый проект в массив
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - указатель на переменную с количеством записей
  - ``newItem`` - указатель на новую запись для добавления
- **Возвращает**: 1 при успехе, -1 при достижении максимума

## 6. int makeHash(const char* text)

- **Назначение**: Вычисляет хэш-значение для строки
- **Параметры**:
  - ``text`` - входная строка для хэширования
- **Возвращает**: числовое хэш-значение строки

## 7. int findByRange(DataItem* items, int count, double minSize, double maxSize)

- **Назначение**: Ищет проекты в диапазоне размеров репозитория
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - количество записей в массиве
  - ``minSize`` - минимальный размер репозитория
  - ``maxSize`` - максимальный размер репозитория
- **Возвращает**: количество найденных записей

## 8. int sortData(DataItem* items, int count)

- **Назначение**: Сортирует проекты по хэшу названия и отображает результат
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - количество записей в массиве
- **Возвращает**: количество выполненных перестановок при сортировке

## 9. int saveToFile(DataItem* items, int count, char* filename)

- **Назначение**: Сохраняет все записи в бинарный файл
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - количество записей для сохранения
  - ``filename`` - имя файла для сохранения
- **Возвращает**: количество сохраненных записей, -1 при ошибке

## 10. int loadFromFile(DataItem* items, int* count, char* filename)

- **Назначение**: Загружает записи из бинарного файла в массив
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - указатель на переменную с количеством записей
  - ``filename`` - имя файла для загрузки
- **Возвращает**: количество загруженных записей, -1 при ошибке

## 11. int createTest(DataItem* items, int* count, DataItem* newItem, int num)

- **Назначение**: Создает заданное количество тестовых записей со случайными значениями
- **Параметры**:
  - ``items`` - указатель на массив структур
  - ``count`` - указатель на переменную с количеством записей
  - ``newItem`` - указатель на временную структуру для заполнения
  - ``num`` - количество тестовых записей для создания
- **Возвращает**: количество созданных записей, -1 или -2 при ошибках
